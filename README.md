# State Shapes
Border shapes of first-level subdivisions of countries. The color scheme used was created to be used for dark mode design ("white on black").   
I use OSM for this project with custom edits to match needs in Geotastic game where Google Maps is used  for guessing.   
**WORK IN PROGRESS.** Countries with Google street view are prioritized.   
   
## Colors in hex
c5c5c5	state fill   
5c5c5c	state outline   
363636	semi-official part of state or dispusted territory   
    
252525	suggested background color
