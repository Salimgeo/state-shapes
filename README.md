# State Shapes
Border shapes of first-level subdivisions of countries.   
I use OSM for this project with custom edits to match needs in Geotastic game where Google Maps is used  for guessing.   
WORK IN PROGRESS. Countries with Google street view are created first.   
   
## Colors in hex
c5c5c5	state fill   
5c5c5c	state outline   
dedede	semi-official parts of states or dispusted territory
